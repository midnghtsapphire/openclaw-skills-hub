# OpenClaw Skills Hub

A production-grade Python application that compiles all 9,580+ OpenClaw community skills into a single unified catalog.

## Features

- **Complete Skill Catalog**: Browse all 9,580+ OpenClaw community skills
- **Powerful Search**: Full-text search with category and owner filtering
- **Multiple Interfaces**: CLI, REST API, and programmatic access
- **Export Capabilities**: Export catalog to JSON, CSV, or HTML
- **Skill Installation**: Install skills locally for use
- **Statistics**: Comprehensive statistics about the skill ecosystem
